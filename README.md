# 🐍 Snake Game 
![Snake Game](screenshot.png)
A classic Snake Game built using Python and Pygame.

## Features of Snake Game

- Home Screen
- Start & Exit Buttons
- Pause & Resume
- Score System
- High Score using File Handling
- Sound Effects
- Background Image
- Self Collision
- Wall Collision
- Restart Option

## Technologies Used

- Python
- Pygame

## How to Run

1. Install pygame

```bash
pip install pygame
```

2. Run the game

```bash
python snake.py
```
